How to run locally:

1. Open a terminal in this folder: `C:\Users\SOWMIYA PK\Desktop\gowtham`
2. Install dependencies:

```powershell
npm install
```

3. Start the dev server:

```powershell
npm start
```

The site should open at http://localhost:3000.
